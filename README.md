# AVLA-APP
> Este es un app reto desarollada para AVLA
## 💻 Pre-requisitos

* Debe de tener instalado `NODE JS`
* Debe de tener instalado `ANGULAR 12`

## 🚀 Instalando AVLA-APP

Para instalar  AVLA-APP, siga estas etapas:

Windows:

```
NPM INSTALL
```

```
NG SERVE -o
```
